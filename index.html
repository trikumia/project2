Learning HTML and CSS

HTML

HyperText Markup Language. Provides structure to the content appearing on a website, such as iamges, text, or videos

tag description
<header>

contains h1 - h6 heading
<nav>

defines a block of navigation links such as menus and tables of contents
<main>

encapsulate the dominant contnent within a webpage
<footer>

the contnent at the bottom of the subject information. Contains information such as: contact info, copyright info, terms of use, site mapreferences
<section>

Defines elements in a document, such aschapters, headings, or any other area of the document with the same theme
<article>

holds content that make sense on its own such as articles, blogs, comments etc.
<aside>

contains information that is related to the main content, but not required in order to understand the dominant information
<figure>

encapsulates all types of media
<figcaption>

is used to describe the media in <figure>
<video>

used for media files
<embed>
<audio>

Css

Cascading Style Sheets. Language use to style HTML content on a web page.

The Box Model

property description example
content

The content of the box, where text and images appear
width

width of the content

h1 {
width: 20px;
}

height

height of the contnent

h1 {
height: 100px;
}

padding

the amount of space between the content area and the border.

h1 {
padding: 5px;
}

margin

the amount of space between the border and the outside edge of the element

h1 {
margin: 10px;
}

border

the thickness and style of the border surrounding the content area and padding

h1 {
border: 1px solid red;
}
